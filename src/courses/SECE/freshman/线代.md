# 线代

一些经验：
- 课本上的定理证明不考，所以如果觉得吃力可以直接不看
- 25级题型改动（选填占比增大）后，题目难度比往年难一些
- 我的学习方法效率较低，就给大家提供个参考样本：我是平时不听课但都有自己看课本在学（前期是看丘维声的网课但后来因为不想看推导过程就弃了，感觉没啥必要），课后题没怎么做因为套路化的太简单、证明题又做不出，看了3b1b的动画演示（线性变换视角对理解线代很有帮助，虽然对应试而言性价比不高，虽然大一小登还不明白后续学习有没有啥作用，但还是推荐hhh），也有看框框的速成课讲义（但那个太套路化，应付大部分往年题目都没问题，今年的就吃力些，但及格绝对够了），还看了当年线代的8h课程（学起来远远不止8h，b站上有人估计30h）（那个是考研看的hhh我还是太闲了，而且需要原本就有些基础，整体偏难，很多地方太难的我也直接跳过，但看完那个满绩倒是没问题了）我也没想到合适的“性价比最高”的学习方法，还看其他人了
- 附一份自用线代笔记，因为课本的定理都抄下来了，也加了很多网课笔记，没标清楚重点，内容太多，也许只适合不想看课本的同学看看定理合集，或者时间充裕的同学学习
